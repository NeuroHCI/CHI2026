### Prerequisites

- [Node.js](https://nodejs.org/) (v20.10+)
- [Yarn](https://yarnpkg.com/) (v1.22+)

### Getting Started

1. **Install Dependencies:**
   ```bash
   yarn install
   ```

2. **Run Development Server:**
   ```bash
   yarn dev
   ```
   The site will be available at `http://localhost:4321`.

3. **Build for Production:**
   ```bash
   yarn build
   ```
